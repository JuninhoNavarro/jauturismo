Turismo em Jaú — Descubra a Cidade
 
Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de promover o turismo da cidade de Jaú (SP), apresentando seus principais pontos turísticos, opções gastronômicas e roteiros prontos para visitantes.

A proposta é criar uma plataforma simples que incentive moradores e turistas a explorarem o que há de melhor na cidade, valorizando a cultura local e o comércio regional.

Tecnologias Utilizadas

HTML5 – Estrutura das páginas

CSS3 – Design e responsividade
